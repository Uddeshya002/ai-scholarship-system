Full Stack Project

Scholarship Management System

Manage student registrations, scholarship listings, applications, approvals, and payments from one clean workflow built with Node.js, MySQL, HTML, CSS, and JavaScript.

Student Flow

Register, log in, browse scholarships, upload documents, and track application status.

Admin Flow

Verify students, create scholarships, approve or reject applications, and mark payments.

MySQL Ready

Use the included SQL files directly inside MySQL Workbench on your `localhost:3306` server.
